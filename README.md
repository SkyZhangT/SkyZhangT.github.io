# Sky's Homepage

This is Tianyang's home page project. The goal of this project is not only construct a webpage that contains my resume or personal experiences, my LinkedIn page handled that well. Rather, this is a side project that allows me to share my enthusiasm in photography while trying build a interactive webpage and efficient backend file system.

## Project Planning

This project have multiple stages:<br>

1. Basic HTML/CSS practice. This stage is mainly for me to pickup HTML/CSS syntax and web design skills. (Finished)
   - index page. (Finished)
   - Resume/CV page (Finished)
2. React JS and Material-UI learning. (Finished)
   - Learn basics about React (props, sates, hooks). (Finished)
   - Learn basics about material up (CSS, makeStyle, different tags, animation). (Finished)
   - Learn basics about React-Router (Router, Link, Switch). (Finished)
   - Learn basics about Redux. (In Progress)
3. Front end of the photo sharing webpage. (In progress)
   - Scroll-loading page. Dynamically loads new photo/text while client scrolling down. (Finished)
   - Fetch posts and images from backend server for each post, and display as cards. (Finished)
   - Redesign and implement the CV page in React. (Finished)
   - Enable admin user to make new posts and upload image using HTTP requests. (Finished, first version)
   - Enable user login and sign up. (In Progress)
